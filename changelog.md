## [1.0.6] - 2026-05-20

### Added
- Added a seamless 1-Click Auto Update system. The application can now automatically download the latest `.exe` from GitHub releases, install it silently, preserve all user settings, and automatically restart.
- Added a "View Changelog" button to the update notification banner, which opens the release notes in the default web browser.

### Fixed
- Fixed a bug where opening the Settings menu would cause the application to crash due to a missing UI component dependency.
