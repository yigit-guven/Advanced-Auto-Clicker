## [1.0.5] - 2026-05-19

### Added
- Added "Slow Type Text" mode to automatically write a given string over a designated period of time, calculating character delay to avoid lag.
- Added a multiple-profile Prefab system, allowing users to save and easily switch between an unlimited number of custom sequence setups.
- Added full Windows Uninstaller integration, registering the app in the "Apps & Features" menu and providing clean self-deletion.
- Added professional desktop features: System Tray Integration, Application Logging, Single-Instance Protection, and "Run on Startup" configuration in Settings.
- Added right-click context menu to the profile dropdown for quick Rename, Delete, and Add Empty operations.
