# User Guide - Advanced Auto Clicker

## 🚀 Getting Started
1. Run `python main.py`.
2. The **Dashboard** will appear. This is where you manage your click sequence.
3. Move your mouse to any location and press **F7** to add a "Node" at that position.
4. You will see a **Cyan Circle** appear on your screen at that location.

## ⚙️ Configuration
For each node in the dashboard, you can:
- **Action**: Choose between Left Click, Right Click, Double Click, Key Press, or Hold Key.
- **Delay**: Set the wait time (in seconds) before this action is performed.
- **Key**: Specify the key to press or hold (e.g., 'a', 'space', 'enter').

## 🎮 Controls
- **F6**: Start or Stop the auto-clicking sequence.
- **F7**: Capture the current mouse position as a new node.
- **F8**: Clear all nodes and reset the sequence.

## ✨ Features
- **Visual Overlays**: Always know where your clicks will land.
- **Active Tracking**: The current node being executed will glow **Pink**.
- **Sequence Logic**: Actions are performed in the order they were added.
- **Fail-safe**: Move your mouse to any corner of the screen to emergency stop.
